# SecureSense-LCD-Based-Smart-Intrusion-Detection-and-Alert-System
SecureSense is an Arduino-based security system that uses a PIR sensor to detect human movement. When motion is detected, the Arduino activates a buzzer and LED, while a 16×2 LCD displays an alert message and motion count. When no motion is present, the LCD shows a secure monitoring status.

## Components
- Arduino Uno
- PIR Motion Sensor
- 16x2 LCD
- Buzzer
- LED

## Features
- Motion detection
- Audible and visual alerts
- Intrusion counter
- LCD status display
